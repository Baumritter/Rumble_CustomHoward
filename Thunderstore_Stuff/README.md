## Installation Instructions
1. Install MelonLoader
2. Run Game
3. Drop "Mods" and "UserData" Folder in to Rumble Source Directory
4. Run the Game and enjoy the mod

## Overview
- Replaces the Level Controls of Howard with Controls that support more than 3 Levels
- Adds a framework for adding your own movesets
- Mod comes with a few Movesets pre-installed

## Controls
- "Prev" Button: Cycles downwards through the movesets 
- "Next" Button: Cycles upwards through the movesets 
- "Reload" Button: Re-reads the files in the Moveset-Folder 

## ModUI Implementation
- Open via "F10"
- Description: Shows what Movesets are loaded. If more than 10 movesets are loaded it will cycle through on a 3 second interval.
- Reload all Movesets: If set to "true" it will perform the same action as the "Reload" Button. Will set itself to false after 0.5 seconds.
